# Porsche 911 Premium Website


## Features

- **Premium Dark Mode Design** - Automotive luxury aesthetic with Guards Red accents
- **Glassmorphism Navigation** - Frosted glass navbar with smooth scroll effects
- **Responsive Layout** - Optimized for mobile, tablet, laptop, and desktop
- **Smooth Animations** - Fade-ins, parallax effects, and micro-interactions
- **7 Key Sections**:
  - Hero with large "911" typography
  - Heritage timeline
  - Performance specifications
  - Design philosophy
  - Models lineup (Carrera, Turbo, GT3)
  - Experience gallery
  - Premium footer

## Design System

- **Colors**: Deep blacks, racing grays, Guards Red (#dc0a28)
- **Typography**: Geometric sans-serif with strong hierarchy
- **Spacing**: 8px system for consistent rhythm
- **Animations**: 0.2-0.4s smooth transitions
- **Border Radius**: 16-24px rounded corners

## Technologies

- Semantic HTML5
- Modern CSS (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript
- CSS animations and transforms
- Intersection Observer API


## Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)

## Performance

- Optimized animations
- Lazy loading support
- Reduced motion support for accessibility
- Hardware acceleration for smooth 60fps
